# Strona niespodzianka

Statyczna strona dla GitHub Pages.

## Pliki
- `index.html`
- `style.css`
- `script.js`
- `image-pools.json`
- `suchary.json`

## Działanie
- Pokazuje aktualną datę i okazję.
- Pokazuje suchara.
- Pokazuje laurkę z wybranych kategorii e-kartki.
- Przycisk `Zgeneruj` losuje nową laurkę i nowego suchara.
